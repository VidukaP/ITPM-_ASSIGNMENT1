# Project Description
Automated testing for Singlish to Sinhala conversion on Swift Translator using Playwright

# Installation
1. Clone the repository or extract the folder.
2. Run `npm install` to install dependencies.
3. Run `npx playwright install` to set up browser. (Only enabled Chrome)


# Execution  
1. To run all the tests
        npx playwright test

2. To view the HTML report
        npx playwright show-report

3. To run all tests in headed mode (To see the browser)
        npx playwright test --headed
